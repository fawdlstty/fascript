## 特点

- 托管任务（Task）可跟踪记录、暂停、恢复、强制结束
- 流程只需要编写基本逻辑，其他异常处理逻辑通过描述符来完成，简化代码

## 注解

- @atomic
  + 代表task为原子任务，无法执行暂停、中断等操作
